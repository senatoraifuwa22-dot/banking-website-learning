# Screenshot Instructions

Capture and store three screenshots for documentation:

1. **Home** page
2. **Dashboard** page
3. **Transfer/Receipt** flow

Guidelines:
- Target width: **~1200px** for each screenshot.
- Save files in this folder using these names:
  - `home.png`
  - `dashboard.png`
  - `transfer.png`

Referencing in Markdown:
```markdown
![Home](screenshots/home.png)
![Dashboard](screenshots/dashboard.png)
![Transfer or Receipt](screenshots/transfer.png)
```
