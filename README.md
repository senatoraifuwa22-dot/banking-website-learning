# Banking Website (Learning Project)

Welcome! This is a beginner-friendly online banking UI built with simple, static assets. Use this README to explore the live demo, see screenshots, and understand how the project is organized.

## Features Summary
- **Static site:** Served as plain HTML/CSS/JS for easy hosting.
- **Vanilla JavaScript:** No frameworks required; just open in a browser.
- **Contact officer flow:** Shows a modal with a confirmation toast when you reach out, keeping feedback clear for users.

## Project Structure (Quick Overview)
- `index.html` – Landing page entry point.
- `banking-website/` – Core static assets (HTML, CSS, JS, images).
- `banking-website/shared/docs/screenshots/` – Project screenshots used above.
- Additional assets may be added over time, but the structure stays intentionally simple for learning.
